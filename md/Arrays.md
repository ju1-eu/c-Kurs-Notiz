# Arrays (Variablenketten)

<!--7-Mrz-20-->

- Bezeichner (Name)
- Datentyp
- Länge (Anzahl der Elemente)

```
    int iValue = 0;
    int iArray[3];
    iArray[0] = 200;
    iValue = iArray[0];
    int iArray2[] = {10, 20 ,30};

    // Hinweise
    iArray    = Name allein, Adresse des Arrays
    iArray[4] = Index beginnt bei 0, 5. Eintrag
    iArray[5] = Bereichsüberschreitung, letzter Eintrag 4
```
